Dungeon Lord

This project was for my mobile game module in our final year, we were tasked with designing and creating a mobile game using Unity.

My game was inspired by the card game Boss Monster. The game is a turn based game where every round a random selection of enemies spawn in the entrance of your dungeon, using the gold you collect from slaying enemies you need to build rooms in your dungeon to protect you from the attacking enemies.
